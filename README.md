# C-BLOCK Lifestyle Brand Website

A modern, responsive website for C-BLOCK - a fashion + community + access platform that empowers young people through style, connection, and opportunity.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Scroll-triggered animations and hover effects for enhanced user experience
- **Modern UI**: Clean, contemporary design with card-based layouts and intuitive navigation
- **Multi-Page Structure**: Comprehensive site including homepage, clothing, community, join, about, and contact pages
- **Brand Identity**: Consistent typography (Montserrat & Inter fonts) and color scheme

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Advanced styling with custom animations and responsive design
- **JavaScript**: Scroll-triggered animations using Intersection Observer API
- **Google Fonts**: Montserrat and Inter for typography

## Pages

- **Homepage** (`index.html`): Hero section, welcome content, and feature highlights
- **Clothing** (`clothing.html`): Product showcase and R200 store information
- **Community** (`community.html`): Membership benefits and community features
- **Join** (`join.html`): Team opportunities and application process
- **About** (`about.html`): Brand story, mission, and company information
- **Contact** (`contact.html`): Contact details, social media links, and FAQ

## Setup & Installation

1. Clone or download the project files
2. Open `index.html` in your preferred web browser
3. Navigate through the site using the responsive navigation menu

For local development or hosting:
- No build process required - pure HTML/CSS/JS
- Can be hosted on any static web hosting service (Netlify, Vercel, GitHub Pages, etc.)

## Usage

- **Navigation**: Use the hamburger menu on mobile devices
- **Animations**: Scroll through pages to trigger smooth entrance animations
- **Responsive**: Resize browser window or view on different devices to see responsive design
